Sistema de Monitoreo de Servidores Linux

Desarrollado como proyecto final para la asignatura de Administración de Sistemas

Este proyecto implementa un sistema de monitoreo para servidores Linux que supervisa en tiempo real el uso de CPU, memoria, disco y red. Envía alertas por correo electrónico cuando los recursos superan umbrales predefinidos.

La aplicación utiliza scripts en Bash para la recolección de datos y Python para el procesamiento y visualización. Los datos se almacenan en una base de datos SQLite y se presentan a través de una interfaz web simple construida con Flask.

Tecnologías utilizadas:

Python Bash Flask SQLite Systemd Cron

Automatización de Despliegue con Docker

Proyecto personal para simplificar el despliegue de aplicaciones web

Desarrollé un conjunto de scripts y configuraciones para automatizar el despliegue de aplicaciones web utilizando Docker y Docker Compose. El sistema incluye contenedores para la aplicación, base de datos, servidor web y caché.

La solución implementa un flujo de trabajo completo desde el desarrollo hasta la producción, con entornos separados y configuración mediante variables de entorno. Incluye respaldo automático de datos y rotación de logs.

Tecnologías utilizadas:

Docker Docker Compose Nginx PostgreSQL Redis Bash

API RESTful para Gestión de Inventario

Desarrollado como parte de un proyecto colaborativo universitario

Diseñé e implementé una API RESTful para un sistema de gestión de inventario. La API permite realizar operaciones CRUD sobre productos, categorías, proveedores y transacciones de inventario.

El proyecto incluye autenticación JWT, validación de datos, documentación con Swagger, pruebas unitarias y de integración, y está desplegado en un entorno containerizado.

Tecnologías utilizadas:

Python FastAPI PostgreSQL SQLAlchemy Docker JWT Pytest
